# flask-chatbot
Built on python 3.7
Flask==0.12.3
chatterbot==0.8.4
SQLAlchemy==1.2.0

#### A web implementation of chatterbot using Flask.

## Local Setup:
 1. Open command prompt and locate folder. run 'pip install -r requirements.txt'
 2. Run *train.py*
 3. Run *run.py*
 4. Demo will be live at http://localhost:5000/
